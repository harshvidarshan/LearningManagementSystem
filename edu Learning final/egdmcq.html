MCQs of Engineering Graphics & Drawing

1.The commands Erase, Copy, Mirror, Trim, Extend, Break etc belongs to which tool
(a)Layer tool bar
(b)Drawing tool bar
(c)Modify tool bar
(d) None
Answer:
Option c)Modify tool bar
2.The commands Donut, Block, Spline, Polygon, and Arc etc belong to which tool bar?
(a)Layer tool bar
(b)Modify tool bar
(c)Draw tool bar
(d)None
Answer:
Option c)Draw tool bar
3.The command which works on two lines or a single polyline to create a beveled edge is __________.
(a) Fillet
(b) Chamfer
(c) Stretch
(d) None
Answer:
Option b)Chamfer
4.The command which is used to create a round corner between two lines is __________.
(a)Fillet
(b)Chamfer
(c)Stretch
(d)None
Answer:
Option a)Fillet
