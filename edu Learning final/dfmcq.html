MCQs of Fundamentals of Digital Systems and logic families

1. The highest decimal number that can be represented with 9 binary digits is
(a) 512
(b) 511
(c) 255
(d) 256
Answer:
Option b)511
2. Perform 10111.101 + 110111.01
(a) 1010110.11
(b) 1001110.111
(c)1101010.101
(d) 1011101.101
Answer:
Option b)1001110.111
3.Subtract (37)8 from (53)8.
(a) -15
(b) -16
(c) 15
(d) 14
Answer:
Option d)14
4. Convert decimal number 675.625 to hexadecimal.
(a) 2A3.A
(b) 2A3.10
(c) 2A34
(d) 2103.A
Answer:
Option a)2A3.A
5.Convert binary number 110110010 to gray code.
(a) 10110011
(b)1000111
(c)11001100
(d) 101101011
Answer:
Option d)101101011
6. Following is not invalid code in XS-3.
(a) 0010
(b) 1100
(c) 1011
(d) 1101
Answer:
Option c)1011
7.Convert XS-3 code 0101 0111 to BCD code
(a) 0010 0100
(b) 1000 0111
(c) 0011 1001
(d) 1000 1001
Answer:
Option a)0010 0100
